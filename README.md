# Metis alpha
**PMO Intelligence Platform** — Aligning meaning before decisions collapse.

## セットアップ

```bash
npm install
npm run dev
```

## デプロイ（Vercel）

1. このフォルダをGitHubリポジトリにプッシュ
2. [vercel.com](https://vercel.com) でGitHubリポジトリをImport
3. 設定はデフォルトのままDeploy
4. URLが発行される

## 機能

- **Dashboard** — Static × Dynamic 二軸のプロジェクト健全性スコア
- **Gravity View** — 依存構造の重力可視化 + Drift Chart
- **Semantic Ghost** — ⌘K でSemanticSpaceに自然言語で問い合わせ（Anthropic API）
- **プロジェクト作成** — Static/Dynamic 10ボックス定義 + ファイルドロップ

## 注意

Semantic Ghost（AIチャット機能）はAnthropic APIキーが必要です。
